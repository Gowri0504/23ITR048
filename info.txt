## 2. How to Run

### Prerequisites
- Node.js installed (v16.13.0 used for development).
- Internet connection (for external Logging API).

### Running the Backend
1. Open a terminal.
2. Navigate to the backend directory:
   ```bash
   cd backend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   node server.js
   ```
   The server will run on `http://localhost:5001`. (If port 5001 is busy, it will use the next available port or you can set `PORT` in a `.env` file).

### Running the Frontend
1. Open a new terminal.
2. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:3000` (or `http://localhost:3001` if 3000 is busy). 
   **Note**: The frontend is configured to talk to the backend on `http://localhost:5001/api`.
